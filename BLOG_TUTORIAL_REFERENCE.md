# 教程与样例文章说明

这些文章目前保留在站内，作为 Fuwari / Astro 博客功能参考使用，但不会在首页文章列表中展示。

## 保留原因
- 以后改博客功能、排版或内容格式时，可以直接参考现成例子。
- 避免把教程内容误当成正式博客文章展示在首页，破坏个人站气质。

## 当前保留的教程/样例文章

### 1. `src/content/posts/guide/index.md`
- 标题：`Simple Guides for Fuwari`
- 用途：整体使用说明，包含文章 frontmatter、目录结构等基础约定。
- 适合查：新增文章、封面图、标签、分类这些基本问题。

### 2. `src/content/posts/markdown-extended.md`
- 标题：`Markdown Extended Features`
- 用途：扩展 Markdown 功能示例。
- 适合查：GitHub 仓库卡片、admonition、扩展语法如何写。

### 3. `src/content/posts/expressive-code.md`
- 标题：`Expressive Code Example`
- 用途：代码块展示能力示例。
- 适合查：代码高亮、终端框、标题栏、行号、frame 样式效果。

### 4. `src/content/posts/markdown.md`
- 标题：`Markdown Example`
- 用途：基础 Markdown 示例文章。
- 适合查：标题、列表、引用、代码块、普通文章正文结构。

### 5. `src/content/posts/video.md`
- 标题：`Include Video in the Posts`
- 用途：嵌入视频示例。
- 适合查：YouTube / Bilibili iframe 的写法。

### 6. `src/content/posts/draft.md`
- 标题：`Draft Example`
- 用途：草稿机制示例。
- 适合查：如何使用 `draft: true` 隐藏未完成文章。

## 首页展示策略
首页当前已隐藏这些教程/样例文章：
- `guide`
- `markdown-extended`
- `expressive-code`
- `markdown`
- `video`

这样可以保留它们的参考价值，同时让首页只承载真正属于个人站的内容。

## 后续建议
- 等你自己的正式文章数量起来后，可以继续保留这些文章，只作为站内功能参考。
- 如果以后想更彻底，可以把这些文章单独归档到一个“文档 / 备忘”分组，而不是普通博客流。
